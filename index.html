Responsive Web Design (RWD)

Vad är Responsive Web Design? Betyder att layouten ändras baserat på den enhet (mobil, tablet, desktop) som använder sitter på. Samma HTML-element (i stor utsträckning). Brytpunkter definerar i CSS med hjälp av mediaqueries. Mobile-first Är en strategi där man skapar layout för mobilen först och sedan för desktop. Varför? Många hävdar att detta approach fokuserar mer på huvudinnehållet. Du behvöver oftast mindra CSS-regler för mobil och det är sedan enklare att lägga till CSS-regler för desktop än tvärtom. När du skriver CSS för mobil-first använder du min-width i dina mediaqueries istället för max-width
